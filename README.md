# Download MP4 and MP3 from YouTube
Simple UI and fast download MP4 and MP3 from YouTube url.

# Setup
```
Download python 3 and install pip
pip install pytube
pip install youtube_dl
pip install pysimplegui
run with :
cd to your folder
and write
project_2.py
```
